## Split programs in the manner : skeleton, function prototypes and main function in the main c file (`queue2.c`) and the functions in a seperate file (`functions.c`), compile them combined together and run the program.
This task can be done by executing the commands 
```bash
gcc queue2.c functions.c -o queueprogram.out
```
Output file is optional and the command can also be run without explictly giving `-o` parameters, for which the executable will be produced as `a.out`
```bash
gcc queue2.c functions.c
```
